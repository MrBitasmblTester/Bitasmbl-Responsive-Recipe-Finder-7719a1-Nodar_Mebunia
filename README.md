# Bitasmbl-Responsive-Recipe-Finder-7719a1-Nodar_Mebunia

## Description
A beginner-friendly React and CSS project where users can search for recipes and view dynamic cards displaying images, ingredients, and cooking steps. The app emphasizes component structure, responsive layout, and API data rendering.

## Tech Stack
- Next.js
- React
- Tailwind CSS

## Requirements
- Use functional React components and hooks
- Fetch data from a public API and render dynamically
- Implement mobile-responsive design using Tailwind CSS

## Installation
bash
git clone https://github.com/MrBitasmblTester/Responsive-Recipe-Finder.git
cd Responsive-Recipe-Finder
npm install


## Usage
bash
npm run dev


## Implementation Steps
1. Initialize a Next.js app structure.
2. Configure Tailwind CSS for global styling.
3. Create functional components for search and recipe cards.
4. Use React hooks to manage search state and API data.
5. Fetch recipes from a public API and render cards.
6. Add responsive Tailwind classes for mobile layouts.
7. Display images, ingredients, and cooking steps dynamically.
